OpenShift Workshop

This workshop is for a A full developer experience in the cloud - Mastering OpenShift DevSpaces in Enterprise Environments. The deployment provisions an OpenShift cluster with basic customizations on AWS infrastructure.

Cluster Specification

  • 3 Masters Nodes (AWS instance type: m5a.2xlarge)
  • Number of nodes dependant on the size ordered (AWS instance type: m5a.4xlarge)
  • Cluster Version: 4.12.x
  • Authentication Enabled (htpasswd - refer to your provisioning e-mail for details)
  • LimitRanges enabled
  • Let's Encrypt Certificates enabled on Ingress Controllers

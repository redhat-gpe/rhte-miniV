This workshop is designed to provide an OCP (OpenShift Container Platform) cluster running ACS (Advanced Cluster Security) that can host multiple users in a centralized and secured environment. Participants will have hands-on access to the platform, learning how to scan their environment for vulnerabilities, manage security policies, and investigate incidents. The workshop is ideal for hosting events, seminars, or training sessions that require a secure environment for multiple users. Overall, this workshop provides organizations with a unique opportunity to learn about ACS and gain practical experience that can be applied to their own infrastructure, making it more secure.

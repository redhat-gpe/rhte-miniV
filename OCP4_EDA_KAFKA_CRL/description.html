Architecting event-driven apps with OpenShift, CockroachDB, and Kafka

OpenShift Workshop provisions an OpenShift cluster with basic customizations on AWS infrastructure for running hands-on-labs, workshops and similar events where no specialized environment is available.

For specialized workshops (such as Containers and Cloud-Native Roadshow, OpenShift Getting Started Workshops, etc.) please use the dedicated catalog item.

Cluster Specification

  • 3 Masters Nodes (AWS instance type: m5a.2xlarge)
  • Number of nodes dependant on the size ordered (AWS instance type: m5a.4xlarge)
  • Cluster Version: 4.12.x
  • Authentication Enabled (htpasswd - refer to your provisioning e-mail for details)
  • LimitRanges enabled
  • Let's Encrypt Certificates enabled on Ingress Controllers
